QUICK START

Install Chiliz MCP in Claude Desktop

Add blockchain superpowers to Claude in just 2 minutes

1

Download & Build

Open Terminal and run these commands:

# Clone the Chiliz MCP repository
git clone https://github.com/BrunoPessoa22/chiliz-mcp.git

# Navigate to the directory
cd chiliz-mcp

# Install dependencies
npm install

# Build the project
npm run build
2

Locate Config File

Open Claude Desktop's configuration:

# Open config in your default editor
open ~/Library/Application\ Support/Claude/claude_desktop_config.json

# Or use nano/vim
nano ~/Library/Application\ Support/Claude/claude_desktop_config.json
If the file doesn't exist, create it first with: touch ~/Library/Application\ Support/Claude/claude_desktop_config.json
3

Add Configuration

Add this to your config file:

{
  "mcpServers": {
    "chiliz-mcp": {
      "command": "node",
      "args": ["/Users/YOUR_USERNAME/chiliz-mcp/dist/index.js"],
      "env": {
        "CHILIZ_RPC_URL": "https://rpc.ankr.com/chiliz",
        "NETWORK": "mainnet"
      }
    }
  }
}
Replace YOUR_USERNAME with your actual macOS username
4

Restart Claude

Restart Claude Desktop to activate:

Quit Claude Desktop completely
Reopen Claude Desktop
Chiliz MCP is now active!

Troubleshooting & Common Issues

MCP not showing in Claude

• Make sure the path in config is absolute, not relative

• Check that dist/index.js exists after building

• Verify JSON syntax is correct (no trailing commas)

Build errors

• Ensure Node.js 18+ is installed

• Try deleting node_modules and reinstalling

• Check npm version with npm --version

Connection errors

• Verify RPC URL is accessible

• Check network connectivity

• Try alternative RPC: https://spicy-rpc.chiliz.com
